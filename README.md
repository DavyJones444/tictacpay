Um das Projekt auszuführen empfiehlt es sich in VSCode die Extension "Live Server" zu installieren.
Dann müsste unten am Ende von VSCode (dort wo auch Ln, Col, UTF-8, etc. steht) ein Button "Live Server" sein. 
Mit dem könnt ihr dann einen kleinen Webserver starten, der, wenn ihr euch in der main.html befindet, euch auch gleich TicTacPay startet.
